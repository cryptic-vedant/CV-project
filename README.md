# 🧘 Yoga Pose Detection using YOLOv8

This project implements a **Yoga Pose Detection system** using the Ultralytics YOLOv8 framework. It detects human body keypoints from images and visualizes poses, making it useful for fitness analysis, posture correction, and AI-based yoga assistants.

---

## 🚀 Features

* 🔍 Detects **human poses using YOLOv8 pose model**
* 📌 Extracts **17 key body keypoints (COCO format)**
* 🖼️ Visualizes poses on input images
* 📊 Provides **keypoint coordinates with confidence scores**
* 🧪 Supports testing on multiple yoga pose images

---

## 🛠️ Tech Stack

* Python 🐍

## 🧠 Key Concepts

* **Pose Estimation**: Detecting human joints (keypoints) from images
* **COCO Keypoints**: 17 standard body joints like nose, shoulders, knees, etc.
* **Confidence Thresholding**: Filters unreliable detections
